# Graph Editor

## Tech Stack

- **Backend**: Node.js + Express (port 4001)
- **Frontend**: React 19 + Vite (port 5174)
- **Styling**: Plain CSS with CSS variables (dark theme)

## Quick Start

```bash
chmod +x start.sh
./start.sh
```

This installs dependencies and starts both the server and client.

Open http://localhost:5174 in your browser.

## Manual Setup

### Backend

```bash
npm install
node server.js
```

### Frontend

```bash
cd client
npm install
npm run dev
```
